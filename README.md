# Titanic EDA Streamlit App

A Streamlit-based Exploratory Data Analysis (EDA) application using the Titanic dataset.

## Features
- Upload CSV dataset
- Dataset preview
- Dataset information
- Missing value analysis
- Heatmap visualization
- Exploratory Data Analysis (EDA)

## Technologies Used
- Python
- Pandas
- Streamlit
- Matplotlib
- Seaborn

## Dataset
Titanic Dataset
